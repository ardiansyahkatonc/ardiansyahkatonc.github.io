# VISUAL STANDARD v5.0

**Status**: Active
**Version**: 5.0
**Target**: Premium AI-Native Product Studio Experience (Vercel/Linear tier)

## Philosophy
kat.on is an AI-Native Product Studio, not a standard software house or a blog. Every visual element must project authority, intelligence, and high-end engineering. ARIA is our flagship product.

## 1. Color System
- **Background**: `#F8FAFC` (Slate 50) - A premium off-white that reduces eye strain.
- **Primary**: `#0F172A` (Slate 900) - Deep, executive contrast.
- **Accent**: `#2563EB` (Blue 600) - For critical calls to action.
- **Success**: `#16A34A` (Green 600) - For statuses.
- **Border**: `rgba(15, 23, 42, 0.08)` - Barely visible, relying on whitespace over boxes.

## 2. Typography
Typography is the primary UI element.
- **Hero Display**: 64-72px
- **Section Headers**: 40px
- **Card Titles**: 28px
- **Body Content**: 18px
- **Captions & Labels**: 14px

Bilingual content strategy: Indonesian is primary (bolder/larger), English is secondary (muted/smaller) placed directly beneath.

## 3. Shape & Space
- **Radius**: `24px` global radius for all cards and containers to create a modern, friendly, yet highly sophisticated look.
- **Shadows**: Soft, diffused elevations for interactivity, avoiding harsh drop-shadows.
- **Spacing**: Looser tracking. Whitespace is heavily utilized.

## 4. Micro-Animations
- Smooth scrolling.
- Native CSS fade-ins (`reveal`).
- Hover lifts (`translateY(-4px)`).
- Sticky backdrop blurs.

## 5. UI Illustrations
No static screenshots or generic vector art. Dashboards and interfaces are built using abstract CSS shapes, gradients, and glass effects to look like functional, premium software.
